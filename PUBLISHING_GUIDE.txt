===============================================
  BizPilot AI - 上架发布指南
  Publishing Guide for App Store & Google Play

一、项目概览
==========
应用名称: BizPilot AI
定价模式: 订阅制 $99/月 (含7天免费试用)
年付方案: $799/年 (节省33%)
目标平台: iOS (App Store) + Android (Google Play)
技术栈: React Native + Expo (EAS Build)


二、开发环境配置
===============

1. 安装依赖:
   $ npm install -g expo-cli eas-cli
   $ cd BizPilotAI
   $ npm install

2. 本地开发运行:
   $ npx expo start
   (使用 Expo Go 扫码在手机上预览)

3. 配置 EAS:
   $ eas login
   $ eas build:configure


三、上架 Apple App Store (iOS)
==============================

步骤1: Apple Developer 账号
- 注册 https://developer.apple.com ($99/年)
- 创建 App ID: com.bizpilot.ai
- 创建 Provisioning Profile

步骤2: App Store Connect 配置
- 登录 https://appstoreconnect.apple.com
- 创建新 App，填写:
  * 名称: BizPilot AI
  * Bundle ID: com.bizpilot.ai
  * SKU: bizpilot-ai-001
  * 主要语言: English (U.S.)
  * 主分类: Business
  * 副分类: Productivity

步骤3: 配置内购 (In-App Purchases)
- 在 App Store Connect > 内购 中创建:
  * 自动续期订阅组: "BizPilot Pro"
  * 订阅1: bizpilot_pro_monthly_99
    - 价格: $99.00/月
    - 免费试用: 7天
  * 订阅2: bizpilot_pro_annual_799
    - 价格: $799.00/年
    - 免费试用: 7天

步骤4: 构建 & 提交
   $ eas build --platform ios --profile production
   $ eas submit --platform ios

步骤5: App Store 审核清单
- [ ] 截图 (6.7", 6.5", 5.5" 三套尺寸)
- [ ] App 描述 (中英文)
- [ ] 隐私政策 URL
- [ ] 支持 URL
- [ ] App 预览视频 (可选但推荐)
- [ ] 年龄分级: 4+
- [ ] 审核备注: 提供测试账号


四、上架 Google Play Store (Android)
=====================================

步骤1: Google Play 开发者账号
- 注册 https://play.google.com/console ($25 一次性)

步骤2: 创建应用
- 应用名: BizPilot AI
- 包名: com.bizpilot.ai
- 分类: 商务
- 内容分级: 填写问卷

步骤3: 配置订阅 (Google Play Billing)
- 在 Play Console > 获利 > 订阅 中创建:
  * 订阅: bizpilot_pro_monthly_99
    - 基础方案: $99.00/月
    - 免费试用: 7天
  * 订阅: bizpilot_pro_annual_799
    - 基础方案: $799.00/年
    - 免费试用: 7天

步骤4: 构建 & 提交
   $ eas build --platform android --profile production
   (生成 .aab 文件)
   $ eas submit --platform android

步骤5: Google Play 审核清单
- [ ] 截图 (手机 + 7寸平板 + 10寸平板)
- [ ] 精品详情描述
- [ ] 功能图片 (1024x500)
- [ ] 隐私政策 URL
- [ ] 数据安全表单填写
- [ ] 内容分级问卷
- [ ] 目标受众选择


五、RevenueCat 订阅管理配置
============================

1. 注册 https://www.revenuecat.com (免费到 $2,500 MRR)

2. 创建项目 "BizPilot AI"

3. 配置平台:
   - iOS: 上传 App Store Connect Shared Secret
   - Android: 上传 Google Play Service Account JSON

4. 创建 Entitlement:
   - ID: "pro"
   - 关联两个订阅产品

5. 创建 Offering:
   - ID: "default"
   - 添加 Monthly 和 Annual 包

6. 获取 API Keys:
   - iOS API Key -> 替换到 app.json 和 subscription.js
   - Android API Key -> 替换到 app.json 和 subscription.js


六、必须替换的占位符
====================

在代码中搜索并替换以下内容:

  YOUR_REVENUECAT_IOS_API_KEY     -> RevenueCat iOS 公钥
  YOUR_REVENUECAT_ANDROID_API_KEY -> RevenueCat Android 公钥
  YOUR_EAS_PROJECT_ID             -> Expo EAS 项目 ID
  YOUR_APPLE_ID                   -> Apple ID 邮箱
  YOUR_APP_STORE_CONNECT_APP_ID   -> ASC 应用 ID
  YOUR_APPLE_TEAM_ID              -> Apple Team ID
  google-service-account.json     -> 放入 Google Play 服务账号密钥文件


七、设计资源需求
===============

上架前需要准备以下设计素材:

1. App 图标:
   - iOS: 1024x1024 PNG (无圆角，系统自动裁切)
   - Android: 512x512 PNG (自适应图标需前景和背景图)
   - 放入 assets/ 目录:
     * icon.png (1024x1024)
     * adaptive-icon.png (1024x1024, 前景)
     * splash.png (1284x2778, 启动画面)
     * favicon.png (48x48, web)

2. App Store 截图 (使用 Figma/Sketch 制作):
   - iPhone 6.7" (1290x2796) x 6张
   - iPhone 6.5" (1242x2688) x 6张
   - iPad 12.9" (2048x2732) x 6张 (如果支持)

3. Google Play 截图:
   - 手机: 1080x1920 x 8张
   - 7寸平板: 1200x1920 x 8张
   - 功能图片: 1024x500

九、后端服务建议
===============

此 App 需要后端 API 支持。推荐架构:

1. 后端框架: Node.js + Express 或 Python + FastAPI
2. 数据库: PostgreSQL (业务数据) + Redis (缓存)
3. AI 服务: OpenAI GPT-4 API 用于 AI Assistant
4. 认证: Firebase Auth 或 Supabase Auth
5. 部署: AWS / GCP / Vercel
6. 文件存储: AWS S3 / Cloudflare R2
7. 支付 Webhook: RevenueCat Server Notifications

推荐快速启动: 使用 Supabase (数据库+认证+存储) + Vercel (API)


十、重要合规事项
===============

1. 隐私政策: 必须提供，包含数据收集、使用、存储说明
2. 服务条款: 必须提供，包含订阅条款和退款政策
3. GDPR: 如面向欧盟用户，需符合 GDPR 要求
4. 订阅说明: Apple 和 Google 都要求在 App 内清晰展示:
   - 订阅价格
   - 订阅周期
   - 免费试用期限
   - 自动续费说明
   - 取消方式
